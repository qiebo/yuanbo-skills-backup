# 顶层设计方法库（结构经验与使用边界）

> 本文件保留 V1.0 中提炼的教育方案结构经验。样本名称与表述仅用于说明设计方法，不代表当前事实，也不得脱离原始证据直接写入新项目。
>
> 原始说明：本文件提炼自一批已交付、经客户确认的教育方案/申报书（**共 19 份金标准样本**：原 13 份 + 缺口补样 6 份；覆盖小学/初中/高中，主题含科创中心、实验室、工程教育、美育融合、课程建设、标准申报书、预算配置、采购协议、定位汇报稿）。
> **样本规模与缺口说明**：从工作目录 218 份有效 docx 中筛选终稿。经全量分类盘点，**职校/职教、劳动教育 两类在语料中无终稿，属真实缺口**——本库不对这两类编造范式，仅以"待补"标注；其余类别（含航空航天，但多为采购/协议型而非建设方案）均有代表样本。公式与方法已在新老样本上交叉验证成立。
> 目标：使本技能产出的方案，在顶层设计的"新颖度、高端感、学校贴合度"上对齐这些金标准。
> 用法：进入阶段二（顶层设计）前通读；产出概念词与结构后，用文末自检清单过一遍。

## 一、顶层设计通用公式

一份达到金标准的顶层设计，几乎都满足以下五个要素的组合：

```
[记忆点概念词] ＋ [结构隐喻] ＋ [学校基因钩子（校训精神＋校史文脉＋建设优势＋未来方向）] ＋ [差异化定位；仅在有证据时使用首创性表述] ＋ [分层/分期逻辑]
```

| 要素 | 作用 | 反面（避免） |
|------|------|------|
| 记忆点概念词 | 一句话让人记住项目 | 空泛的"智慧""未来""一体化"堆叠 |
| 结构隐喻 | 用对仗/数字把复杂架构讲清楚 | 罗列功能模块而无主线 |
| 学校基因钩子 | 证明"这是为这所学校量身定制"——须综合校训精神、校史文脉、建设优势、未来方向四维因子提炼，并以深入调研为前提 | 套用通用模板，换校名即可复用；或只贴校训、忽略校史/优势/方向 |
| 差异化定位 | 说明与同类项目的具体差别；首创性结论须有可核验比较范围 | 只说“领先”“先进”或无证据宣称“首个” |
| 分层/分期逻辑 | 让建设路径可信、可落地 | 一次堆砌所有内容，无递进 |

**真实样本对照：**

- 位育中学：`智芯致微` ＋ `芯片+X 多尺度生态` ＋ `识芯→造芯→用芯 闭环` ＋ `全国首个高中阶段芯片生物应用实验室` ＋ `总体规划、分期实施`
- 锡师附小：`诚勇启新` ＋ `高校-研究院-产业-中小学 四位一体` ＋ `诚勇校训` ＋ `降维不减配` ＋ `三阶递进课程`
- 湖南师大附中：`一核两翼` ＋ `虚实结合（实=VEX / 虚=AIGC）` ＋ `学-练-赛-创一体化` ＋ `以虚助实、虚实共生` ＋ `三层空间递进`
- 南洋中学：`智联智造` ＋ `串珠成链` ＋ `百年南货站地缘文脉` ＋ `从南站货运到全球供应链的古今转译` ＋ `识技—研技—用技 闭环`
- 成都七中：`蜀道融界·智启未来` ＋ `三层四翼（三层阶梯课程×四翼支撑体系）` ＋ `审是迁善校训×墨池书院文脉×竞赛顶尖优势×对标省级示范方向` ＋ `对标教育部的四川省AI教育示范标杆` ＋ `贯通设计、分层实施`
- 市二中学：`戏影智境` ＋ `以戏剧为载体的具身智能人机共演` ＋ `务本传统×影视戏剧品牌×人机共演` ＋ `国内校园首个聚焦"人机共演"的戏剧排演空间` ＋ `普惠-特色-创新三级课程`
- 进才中学：`地球与宇宙科学（拳头项目）` ＋ `四维认知底座＋三径工程驱动＋数智寰宇实践中心` ＋ `进步即成才理念×地学竞赛品牌×浦东三大先导产业×学校级平台` ＋ `扬长型学校级科学教育平台` ＋ `由点到面、由优到强、由特色到品牌`
- 金华外国语：`语言+工程 / 未来工程师` ＋ `认知-实践-表达三位一体＋高中-高校-企业三级共建＋六大功能空间` ＋ `国际化基因×语言教学优势×新能源/具身智能产业` ＋ `外语见长、工程硬核的差异化办学品牌` ＋ `普惠-进阶-拔尖梯度`
- 西南位育（预算型）：`无概念词` ＋ `按能力模块分类（智慧能源/风能/氢能/碳捕集…）` ＋ `建设优势即课程方向` ＋ `每条"市场询价"依据` ＋ `无分层（纯概算）`
- 宁波科创（采购型）：`无概念词` ＋ `设备清单＋技术参数＋配套课时资源` ＋ `航天火箭工程教育` ＋ `真实型号（超音速气流/马赫环/水动力火箭/openRocket）` ＋ `无分层`

## 二、概念词提炼五法

**法一：学校基因综合提炼法**（最推荐，差异化最强，须以深入调研为前提）
学校的"基因"不止校训一维，须综合四维因子提炼项目精神内核，且**先调研、后设计**：
- **校训精神**：如锡师附小"诚勇" → `诚勇启新`、`诚于科学、勇于创新`
- **校史文脉**：如南洋中学毗邻百年南货站 → `智联智造`、"`从南站货运走向全球供应链`"古今转译
- **建设优势**：学校已有场地/师资/设备/课程/合作资源（如位育已有"识芯、造芯"两期基础 → 顺延出 `用芯`，形成 `识芯→造芯→用芯` 闭环）
- **未来方向**：学校"十五五"规划、特色高中/科技高中创建、拔尖创新人才培养等战略意图（须与学校发展规划或会议纪要核对，不臆测）

> 关键原则：**四维取交集、定主线**。优先选"校训精神 × 未来方向"天然吻合且"建设优势"能支撑的主题；若四类因子冲突（如校训偏人文、未来想做硬核科创），以"建设优势 × 未来方向"为主轴，把校训精神作为价值底色融入，而非生硬贴标。

**法二：对仗数字法**
用"数字＋对仗"构筑易记骨架：`一核两翼`、`三阶递进`、`五育融合`、`三级课程`、`四位一体`、`双轮驱动`、`两力八要素`。

**法三：古今转译法**
把学校的历史意象转译为面向未来的学习场景，制造"传承感＋前瞻性"双重张力。
- 南洋：`从南货站的传统转运到未来智造的智能递送`
- 位育：引《中庸》`致广大而尽精微` 诠释"多尺度科创生态"

**法四：虚实/表里法**
适合含软硬件、线上线下的项目：`虚实结合`、`以虚助实、虚实共生`、`软硬一体`、`造-控-测`。

**法五：闭环法**
用"动词链条"呈现培养/建设逻辑的完整性：`识芯→造芯→用芯`、`识技→研技→用技`、`做得出→讲得清→用得上`、`学-练-赛-创`。

> 实操：先完成四维调研、定"学校基因"（法一），再套一个结构框架（法二/四/五），最后用一句话点出"新颖点"。概念词尽量 ≤8 字、有画面、不堆砌"智慧/未来"。

## 四、新颖 / 高端表达技法

**高风险差异化主张**（不是必选项；只有证据覆盖完整时才能使用）：
- `全国首个高中阶段……实验室`、`上海乃至全国首个……`、`区域内率先……`均属于高风险主张，必须明确比较对象、区域、时间范围和检索证据。
- 无法完成充分证明时，改用 `拟形成……`、`面向……重点建设`、`对标……要求`、`探索……路径`。
- `达到……水平`同样需要可衡量标准，不能作为无证据的安全替代词。

**范式/高度词**（克制使用，每篇 1–2 处即可）：
- `新范式`、`新样板`、`标杆项目`、`示范窗口`、`创新高地`、`教育闭环`、`育人新生态`

**时代锚点词**（体现政策敏锐度，须与当期政策一致）：
- `科学教育`、`工程教育`、`教育数字化`、`课程改革`、`新质生产力`等。具体政策名称、规划周期和有效性必须在项目运行时核验

**人文价值词**（避免方案过于"硬核"而缺温度）：
- `科技向善`、`价值引领`、`活化利用历史基因`、`回应真实需求`、`社会责任感`

## 五、学校基因融合五步（以深入调研为前提）

> 此步不可凭空设计。进入顶层设计前，必须在阶段一/阶段二完成对学校的**四维调研**，并整理成"学校基因卡"（见下）。基因融合的质量，取决于调研的深度与证据的扎实度。

**调研四维度（须收集的证据）：**
| 维度 | 收集什么 | 主要来源 |
|------|----------|----------|
| 校训精神 | 校训原文、办学理念、育人目标 | 学校官网 / 招生册 / 校史 |
| 校史文脉 | 建校史、地标建筑与事件、传统特色（百年车站、老校址、历史名人等） | 校史馆 / 地方志 / 区位特征 |
| 建设优势 | 现有场地 / 师资 / 设备 / 课程 / 已建项目 / 合作高校企业 | 会议纪要 / 现状清单 / 实地踏勘 |
| 未来方向 | "十五五"规划、特色高中/科技高中创建、拔尖人才培养、区域定位 | 学校发展规划 / 政府文件 / 关键人访谈 |

**融合五步：**
1. **建基因卡**：把四维度证据填进一张卡片，作为顶层设计的唯一依据（禁止凭印象或套用模板）。
2. **找交集**：在四维中定位与本项目主题的自然交集（如"诚勇"↔科学严谨求真；"南货站"↔物流供应链；"已建识芯造芯"↔用芯闭环）。
3. **定主线**：从四维中选定 1 条主轴（优先"校训精神 × 未来方向"吻合且"建设优势"可支撑者）；冲突时以"建设优势 × 未来方向"为主轴，校训精神作价值底色融入，不生硬贴标。
4. **转译为行为**：把选定基因翻译成项目中的具体动作，而非口号——
   - 锡师：`诚`=尊重事实、严谨求证、忠于原始数据；`勇`=勇敢面对工程挑战、不惧失败反复迭代。
5. **三处呼应**：在 `背景`、`理念`、`意义与价值` 三处反复回扣基因，用项目专属词汇承载（如"智联智造"），杜绝"本项目践行了XX校训"式收尾。

## 六、分层 / 分期逻辑库

- `总体规划、分期实施`（位育）—— 宏大蓝图分多期，每期独立闭环、可示范
- `三阶递进`（锡师、华师一）—— 从普惠到卓越、从素养到拔尖
- `学-练-赛-创`（湖师大）—— 普及入口→集训出口→跨界创造
- `识技—研技—用技`（南洋）—— 知识→研究→应用完整链条
- `纵向贯通、横向融合`（位育课程）—— 纵向承接已有基础，横向打破学科壁垒

## 七、章节结构范式（typology，多家族）

金标准文档按"文类"分家族，顶层设计的**密度与落位**不同。先判文类，再套对应骨架。

### A 类：自由建设方案（汇报/建设方案）
覆盖子类：科创中心、实验室建设、工程教育、农场生态、美育/跨界融合、初中工程。
典型骨架（6 章内）：
```
一、项目背景与建设意义        ← 国家战略 + 地方产业 + 学校四维基因 + 痛点缺口
二、建设目标与顶层设计        ← 【本文件核心】概念词 + 结构隐喻 + 分层逻辑
三、建设内容（空间/装备/平台）  ← 真实设备型号、功能定位、不写参数说明书
四、课程规划                  ← 理念 + 阶梯课程体系 + 真实课时/赛事出口
五、预算分配                  ← 分配原则 + 概算分布（有出处）
（六、预期成效 / 师资保障）
```
子类要点：
- **实验室建设方案**（成都七中范本）：第一章用 `1.1 时代背景 / 1.2 教育背景 / 1.3 学校背景 / 1.4 意义` 四层递进；第二章 `2.1 核心理念 / 2.2 建设目标 / 2.3 建设原则 / 2.4 总体框架`，理念段即概念词+结构隐喻+基因融合的落点。
- **美育/跨界融合**（市二"戏影智境"范本）：以戏剧为载体、具身智能人机共演的跨界转译技法，把非典型科创主题（戏剧）接入科学教育；新颖点常是"国内校园首个…"。
- **初中工程教育**（金华外国语范本）：强"跨界基因"（语言+工程）+ 国际赛事出口（iGEM/Conrad/ISEF），空间用"六大功能空间"叙事串联。

样本：位育、锡师附小、湖南师大附中、交大附中、学军 X-Space、华师一博耕园、成都七中、市二戏影智境、金华外国语。

### B 类：标准申报书（政府固定模板）
模板字段固定（项目基本情况 / 经费概算 / 实施进度 / 保障措施 / 结论），**但内容须把顶层设计注入**：
- `项目基本情况` 中的"项目概况/背景"段，写清：地缘基因 + 概念词 + 结构隐喻 + 新颖点（如南洋把"智联智造"与南货站文脉写进概况）。
- `项目亮点与特色` 段，用 `变革课堂教学/构建特色课程/提升教师能力/整合内外资源` 等小标题承载设计理念（南洋范例）。
- 切忌只填模板字段而丢失"为该校定制"的灵魂。

### C 类：预算配置型（纯概算）
特征：**无概念词、无结构隐喻、无叙事段落**，主体是 `序号/实施内容/规格型号/数量/总额/测量依据` 表格。
规则：
- 分类逻辑即"隐性顶层设计"——按能力模块（智慧能源/风能/氢能/碳捕集…）或空间分区归类，体现建设主线。
- 每条必须标注**测量依据**（市场询价 / 供应商报价单 / 官网），严禁空额。
- 写法极简、正式、缩写化，不展开描述。
样本：西南位育预算。

### D 类：采购 / 合作协议型（航空航天等多为此形态）
特征：以 `设备清单 + 技术参数 + 配套课时资源` 为主体，常含甲乙信息、供货/质量/付款/违约条款。
规则：
- **顶层设计弱**：重点在设备真实型号（超音速气流/马赫环/水动力火箭/openRocket）、技术参数准确、课时资源（如 6 课时电子资源）清晰。
- 附件技术参数描述须专业、可教学（如"硬件实验+软件模拟双向赋能"）。
- 本类不是"建设方案"范式；若客户要的是建设方案，需另起 A 类文档。
样本：宁波科创航天火箭合作协议（注：语料中航空航天几乎均为采购型，缺"航空航天建设方案"金标准，属待补缺口）。

### E 类：定位汇报逐字稿（科学高中/拳头项目）
特征：按 PPT 页序编排、可直接照读；口吻是"学校战略判断"而非"项目推介"。
典型结构（进才范本）：
```
开场与目录 → 政策导向（为什么做）→ 区域发展（同频共振）→ 学校定位（为什么是这个方向）
→ 既有基础（为什么能做好）→ 项目内涵总述（学校级平台）→ 认知底座 → 实施路径 → 预期效应 → 结束语
```
规则：
- 用"回答X个问题"的设问逻辑组织（为什么做/为什么是这方向/为什么能做好/怎么做/什么影响）。
- 立场以确定性判断直接陈述（如"这一项目的关键，在于把既有优势转化为学校级改革的牵引力"）；避免"不是…而是…"式对照——该结构机械感强、易暴露AI写作。
- 概念词/结构隐喻仍要有（四维认知底座、三径工程驱动、数智寰宇实践中心），但服务于"战略判断"而非"方案推销"。
样本：进才中学科学高中定位汇报逐字稿。

### 文类专属语言规范（重要）
- **A/B 类书面方案**：避免"首先…其次…最后"作空洞结构标记；若确为三方面实质枚举（如金华外国语"首先是跨界视野/其次是产业级实践/最后是国际舞台表达"），可用，但须承载真实内容。
- **E 类汇报稿**：可保留口语结构标记（"各位领导/第一/第二"），口吻须"沉稳、克制、专业"，体现"学校战略判断、育人逻辑、区域贡献"三层身份；**不要讲成项目推介**。
- **C/D 类**：极简、表格化、依据标注，不写修饰性语言。

## 八、顶层设计自检清单（产出后必过）

- [ ] 概念词是否 ≤8 字、易记、有画面，而非"智慧/未来"空泛堆叠？
- [ ] 是否钩住了该校**独有**基因（校训精神 × 校史文脉 × 建设优势 × 未来方向 综合提炼），换校名不能复用？
- [ ] 结构隐喻是否自洽，并贯穿背景→理念→内容？
- [ ] 是否有具体差异化定位？如使用首个/领先/标杆等高风险表述，是否已登记完整证据和比较范围？
- [ ] 学校基因是否在背景、理念、意义**三处以上**呼应，且呼应来自调研证据、非凭空贴标？
- [ ] 是否给出了分层/分期逻辑，让建设路径可信可落地？
- [ ] 政策锚点是否为当期有效表述（如"科学教育加法""十五五"）？

> 任一未达标，回到对应技法重写，直至全部通过。

**文类自适应说明**：上述清单对 A/B 类（建设方案、申报书）全量适用；对 C/D 类（预算、采购）**放宽"概念词/结构隐喻"要求**（其顶层设计体现于分类逻辑与依据标注）；对 E 类（汇报稿）概念词须服务于"战略判断"口吻，密度低于 A 类即可。职校/劳动教育两类语料暂无终稿，本库未生成范式，待补。
