# Word 文档样式与版式标准

## 1. 页面

- 默认 A4 纵向；复杂宽表可单独横向；
- 常规页边距 2.5—2.8 cm；
- 封面独立，默认无页眉页脚；
- 目录与正文分页；
- 表格不得超出页面边界。

## 2. 字体层级

默认中文字体：微软雅黑或宋体，按项目要求选择。建议：

- 封面标题：二号至小初，粗体；
- 一级标题：小二/三号，粗体；
- 二级标题：三号/小三，粗体；
- 三级标题：四号，粗体；
- 正文：小四；
- 表格：五号或小五，不低于可读下限；
- 行距：正文 1.5 倍或固定 24—28 磅。

不得使用文字阴影、发光、立体字等装饰。

## 3. 表格

- 表头明确、重复表头；
- 单位放在列名；
- 金额小数位统一；
- 避免整行跨页拆分；
- 列宽按内容分配；
- 表格前后有标题或解释；
- 来源和备注单独列出，不塞入设备名称。

## 4. 图片与图示

- 使用清晰图片；
- 保持纵横比；
- 图片有图题和来源；
- 架构图只呈现正文已有结构，不新增未经核实内容；
- 不使用纯装饰性大图挤压正文。

## 5. QA

文本解析不能替代视觉验收。最终 Word 必须渲染为逐页 PNG 并检查全部页面。
