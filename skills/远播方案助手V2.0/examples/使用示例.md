# 使用示例

## 1. 初始化

```bash
python scripts/init_project.py \
  --name "科创中心建设" \
  --type A \
  --client "某某中学" \
  --output "./某某中学科创中心"
```

## 2. 填写工作底稿

优先完成：

1. `data/需求响应矩阵.csv`；
2. `data/事实证据台账.csv`；
3. `data/课程设备空间映射.csv`；
4. `data/预算明细.csv`；
5. `data/实施验收矩阵.csv`；
6. `07_方案终稿.md`。

## 3. 校验与交付

```bash
python scripts/run_pipeline.py "./某某中学科创中心"
```

打开 `qa/render/page-*.png` 逐页检查。只有视觉检查通过后，才交付 `output/` 中的 Word。
