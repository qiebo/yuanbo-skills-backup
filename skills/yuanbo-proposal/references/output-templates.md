# 方案各阶段输出模板

## 模板说明
以下是六个阶段的标准输出结构。每次使用时，根据具体需求填充内容，结构保持稳定。
占位符用 `【...】` 表示，写作时替换为实际内容。

---

## 阶段一：需求分析报告

**用途**：快速梳理客户需求，形成内部共识，为后续设计提供依据。

```
# 需求分析报告

## 基本信息
- 客户学校：【学校名称】
- 学校类型：【公立/民办 | 普通高中/重点高中/职高/初中等】
- 所在地区：【省市区】
- 接洽时间：【日期】
- 汇报时间：【预计日期】

## 需求背景
【学校当前面临的核心问题或发展诉求，2-3句话概括】

## 核心诉求拆解
| 诉求层面 | 具体内容 | 优先级 |
|---------|---------|-------|
| 显性需求 | 【客户明确提出的要求】 | 高 |
| 隐性需求 | 【客户未明说但实际期望的】 | 中 |
| 痛点问题 | 【当前存在的问题或困境】 | 高 |
| 发展目标 | 【学校希望达到的状态】 | 中 |

## 关键约束条件
- 预算范围：【已知/未知/大约】
- 时间节点：【项目启动/验收时间要求】
- 特殊要求：【政策要求、领导偏好、竞品情况等】

## 竞争态势
【是否有竞争对手参与、学校已有的合作背景】

## 建议切入点
【基于需求分析，建议重点展示远播哪条产品线，核心说服逻辑是什么】
```

## 阶段二：顶层设计框架

**用途**：提炼方案的战略高度，确立核心逻辑，让客户感受到"想得很深"。

```
# 【学校名称】× 远播教育 顶层设计框架

## 一、现状诊断
【用1-2段描述该学校当前在目标领域的现状，点出核心问题】

## 二、目标愿景
【描述合作后学校能达到的状态，要具体、可感知，避免虚话】
例："打造XX市首个AI赋能的生涯教育示范校，为每位学生提供个性化成长路径"

## 三、核心设计理念
【提炼1-2个关键概念，体现方案的思想高度】
例："以生涯为主轴，以数据为驱动，以空间为载体，构建学校发展新生态"

## 四、解决方案框架
### 层次一：【层次名称，如"顶层规划"】
【简述这一层做什么，解决什么问题】

### 层次二：【层次名称，如"系统建设"】
【简述这一层做什么，解决什么问题】

### 层次三：【层次名称，如"落地运营"】
【简述这一层做什么，解决什么问题】

## 五、核心价值主张
【用1段话总结方案对学校的核心价值，这段话要能用在PPT封面或核心页】

## 六、成功标准
| 维度 | 预期成果 |
|-----|---------|
| 【维度1，如学生端】 | 【可量化的成果描述】 |
| 【维度2，如教师端】 | 【可量化的成果描述】 |
| 【维度3，如管理端】 | 【可量化的成果描述】 |
| 【维度4，如品牌端】 | 【可量化的成果描述】 |
```

## 阶段三：方案大纲

**用途**：确定完整方案的章节结构，对齐内容重点后再展开写作。

```
# 【项目名称】方案大纲

## 第一章 背景与现状
1.1 政策环境分析（国家/地方教育政策导向）
1.2 学校现状与发展诉求
1.3 行业趋势与标杆参考

## 第二章 方案总体设计
2.1 设计理念与核心逻辑
2.2 总体架构（示意图）
2.3 建设目标与成功标准

## 第三章 核心解决方案
### 3.1 【模块一名称】
- 功能描述
- 解决问题
- 实施路径

### 3.2 【模块二名称】
- 功能描述
- 解决问题
- 实施路径

### 3.3 【模块三名称（如有）】
...

## 第四章 实施计划
4.1 项目阶段划分
4.2 时间节点安排
4.3 双方职责分工

## 第五章 远播教育介绍
5.1 公司背景与资质
5.2 核心能力与产品体系
5.3 标杆案例（重点呈现同类型学校）

## 第六章 合作模式与投入
6.1 合作方式说明
6.2 项目报价（如已获授权）
6.3 增值服务说明

## 附录
- 相关资质文件
- 参考案例详情
```

## 阶段四：完整文字稿方案

**用途**：方案正文，每章节2-5段完整文字，逻辑清晰、论据充分。

写作规范：
- 每个章节先写"问题/背景"，再写"方案"，最后写"价值/效果"
- 用数据说话：引用远播服务规模、案例成果数据作为背书
- 语气：专业但不生硬，像一个懂行的合作伙伴在说话
- 避免：大量堆砌产品功能介绍，缺乏逻辑连接

---

## 阶段五：汇报PPT大纲

**用途**：基于文字稿提炼PPT页面结构，确定每页主题和核心信息。

```
# 【项目名称】汇报PPT大纲
## 规格：约【X】页，汇报时长【X】分钟

---
【封面页】
标题：【方案主标题】
副标题：远播教育 × 【学校名称】 战略合作提案
日期：【年月】

---
【目录页】共X个部分

---
【Part 1】背景与洞察（X页）
P1. 政策背景
  - 核心信息：【1-2个关键政策点】
  - 视觉建议：【图标/时间轴/政策文件截图】

P2. 学校现状与挑战
  - 核心信息：【提炼的核心问题】
  - 视觉建议：【问题图示/对比图】

---
【Part 2】解决方案（X页）
P3. 方案总览
  - 核心信息：【整体架构，一图说清楚】
  - 视觉建议：【架构图】

P4. 【模块一】
  - 核心信息：做什么 + 解决什么 + 效果
  - 视觉建议：【流程图/产品截图】

...（以此类推）

---
【Part 3】远播实力（X页）
PX. 公司介绍
PX. 核心能力
PX. 标杆案例

---
【Part 4】合作共赢（X页）
PX. 合作模式
PX. 实施计划
PX. 结语/致谢

---
【封底页】
联系信息
```

## 阶段六：PPT内容填充说明

**用途**：逐页列出每张PPT的具体内容，供制作使用。

格式：
```
【第X页】页面标题
- 主标题/核心观点：【文字】
- 正文/要点：
  1. 【第一条】
  2. 【第二条】
  3. 【第三条】
- 数据/案例：【需要呈现的数字或案例简述】
- 备注（演讲要点）：【主讲人说话的重点，不出现在PPT上】
```
