# 审查基准模板（从招标文件提取）

> 用途：精读招标文件文本后，按本模板提取结构化「审查基准」，作为多份响应文件比对的判定依据。
> 凡图片类证据（营业执照、证书、信用截图等）在招标文件中也多为图片，提取时标注「见图片，需核对原件」。

- 采购人 / 采购代理：
- 预算 / 最高限价：
- 递交截止时间：
- 响应（投标）有效期：（天）
- 是否接受联合体：
- 投标保证金：（有/无，金额）
- 评分办法与总分：（综合评分法 / 最低评标价法）

## 二、废标（无效响应）条款清单
> 列出文件具体位置（如「供应商须知前附表 第X项 / 格式四注」）与逐条情形：
1. 未提交保证金（若要求）
2. 逾期 / 未送达 / 未密封 / 电子文档不合规
3. 列入失信 / 税收违法 / 严重违法失信名单
4. 资格条件不符合国家规定或磋商（招标）文件
5. 与其他供应商存在控股 / 管理关系或高管交叉
6. 报价 > 最高限价
7. 响应有效期 < 要求天数
8. 未按格式签字盖章 / 不齐全
9. 其他法定无效情形
- 格式四注：未提供报价明细表 / 明细表合计 ≠ 一览表

## 三、资格性条件
- 营业执照：有效、经营范围匹配
- 财务制度：健全财务会计制度（审计报告 / 报表 / 银行资信）；关注净资产是否为负、成立时间是否过短
- 税收 / 社保：依法缴纳纪录
- 信用：信用中国、中国政府采购网无失信记录截图
- 特定资质：行业许可、软件著作权、厂家授权（如要求）
- 业绩：类似项目数量 / 金额门槛

## 四、符合性 / 形式要求
- 签字盖章齐全（法人 / 代理人、各附表）
- 格式 / 装订 / 份数（正副本、电子版光盘/U盘）
- 响应文件组成完整
- 偏离表已提交

## 五、采购需求与重点条款
> 逐项列出，标注 ★/▲ 等实质性（不可偏离）条款及其证明材料要求：
- 需求1：……
- 需求N（▲）：内容 + 须提供的材料（如团队≥5人名单+专家证明+承诺函、软件界面截图、≥1课时PPT课件）
