# 标书审查详细清单

> 审查时逐项对照，避免遗漏。判定结果记：✅ 已响应 / ❌ 缺失 / ⚠️ 部分 / ？无法判定（图片/原件）。
> 凡标注「？无法判定」的，均因证据以图片承载、文本不可见，须核对盖章正本与随附原件。

## 1. 废标 / 无效响应（最高优先级）
- [ ] 保证金：若要求，是否已提交
- [ ] 递交：是否逾期 / 未送达 / 未密封 / 电子文档不合规
- [ ] 信用：是否列入失信、税收违法、严重违法失信名单
- [ ] 资格：资格条件是否符合国家规定与文件
- [ ] 关联：是否与其他供应商存在控股/管理关系或高管交叉
- [ ] 报价：是否 > 最高限价
- [ ] 有效期：响应有效期是否 < 要求天数
- [ ] 签章：是否按格式签字盖章、是否齐全
- [ ] 其他法定无效情形
- [ ] 格式四注：是否提供报价明细表、明细合计是否 = 一览表

## 2. 资格性
- [ ] 营业执照：有效、经营范围匹配
- [ ] 财务：健全财务会计制度；净资产是否为负、成立时间是否过短（资不抵债风险）
- [ ] 税收 / 社保：依法缴纳纪录
- [ ] 信用截图：信用中国、中国政府采购网无失信（？图片）
- [ ] 特定资质：行业许可、软著、厂家授权（如要求，？图片）
- [ ] 业绩：类似项目数量/金额门槛（合同/中选通知书，？图片）

## 3. 符合性 / 形式
- [ ] 签字盖章齐全（法人、代理人、各附表）
- [ ] 格式 / 装订 / 份数（正副本、电子版光盘/U盘）
- [ ] 响应文件组成完整
- [ ] 偏离表已提交

## 4. 重点（★/▲）条款实质性响应
- [ ] 每项 ★/▲ 条款均有响应
- [ ] 响应满足或优于最低要求
- [ ] 每项均附支撑证据（截图/清单/大纲）
- [ ] 不同章节之间无矛盾

## 5. 价格与一致性
- [ ] 总报价 ≤ 最高限价
- [ ] 单价在分项限额内
- [ ] 税率计算正确
- [ ] 大小写金额一致
- [ ] 无异常低价（恶意低价/围标风险）
- [ ] 付款条件符合

## 6. 技术参数
- [ ] 全部技术参数已响应
- [ ] 声称有检测报告/证书支撑
- [ ] 产品型号全文一致
- [ ] 引用标准现行有效
- [ ] 交付周期合理

## 7. 偏离表诚信（红线）
- [ ] 偏离表声明与正文参数一致
- [ ] 凡「无偏离」项，正文确无超出/不足
- [ ] 存在「无偏离」但参数不符 → 单列 🟠 诚信风险

## 8. 文档完整性
- [ ] 投标/供应商名称全文一致
- [ ] 需签字盖章处均已签署
- [ ] 日期逻辑合理（无未来日期、无早于开标日期）
- [ ] 页码连续
- [ ] 无残留占位符（[X]、[TBD]、[采购人名称] 等）
- [ ] 图片证据（证书/合同/截图/签章）已随附原件或清晰图片

## 9. 常见红旗（fraud indicators）
- [ ] 检测数据与客户值完全相同（造假迹象）
- [ ] 合同缺签名或留白
- [ ] 过期证书当作有效提交
- [ ] 第三方材料无明确授权
- [ ] 不同文件公司名不一致
- [ ] 盗用网络图片（可见水印）
- [ ] 大量复制粘贴的填充内容
